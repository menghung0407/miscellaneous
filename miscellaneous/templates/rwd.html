RWD (Responsive Web Design)

目的

  1. 讓人可以在 不同 裝置上瀏覽 同 一網頁

什麼時候開始?

  1. 2010 年 5 月
  2. 美國網頁設計師 Ethan Marcotte

從技術的角度說

  1. 透過瀏覽器寬度決定網頁版面呈現的一種網頁技術
  2. CSS 能在同一個 HTML 的文本中處理不同螢幕寬度的 style

裝置解析度

  1. 桌上型電腦： 1024px以上
  2. 平板電腦： 720px～1024px
  3. 手機： 320px～720px

常聽到的 CSS framework

  1. Bootstrap
  2. Tailwind
  3. Foundation
  4. Materialize
  5. ...

傳統 VS RWD

結論

即便是使用 RWD 的 framework，也是需要經過設計才能有效果
